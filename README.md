# 파일 복사 프로그램

이 프로그램은 특정 숫자 목록을 입력받아 해당하는 파일들을 자동으로 찾아서 복사하는 도구입니다.

## 기능

- **파일 검색**: xxx03214.jpg, abc1234.png 같은 파일명에서 숫자 부분을 기준으로 파일을 찾습니다
- **다양한 입력 방법**: 직접 텍스트 입력 또는 텍스트 파일 업로드
- **자동 복사**: 원본 폴더에서 저장 폴더로 파일을 자동 복사
- **진행 상황 표시**: 실시간 진행 상황 및 로그 확인
- **중복 파일 처리**: 같은 이름의 파일이 있을 경우 자동으로 이름 변경

## 사용법

### 1. 프로그램 실행
```bash
python file_copier.py
```

### 2. 경로 설정
- **원본 폴더**: 복사할 파일들이 있는 폴더를 선택
- **저장 폴더**: 파일을 복사할 대상 폴더를 선택

### 3. 숫자 목록 입력
두 가지 방법 중 하나를 선택:

#### 방법 1: 직접 입력
- "직접 입력" 라디오 버튼 선택
- 텍스트 영역에 숫자들을 입력 (공백, 줄바꿈, 쉼표 등으로 구분)
```
323
2134
3212
```

#### 방법 2: 텍스트 파일
- "텍스트 파일" 라디오 버튼 선택
- "텍스트 파일 선택" 버튼 클릭하여 .txt 파일 업로드

### 4. 파일 복사 실행
- "파일 복사 시작" 버튼 클릭
- 진행 상황을 실시간으로 확인
- 완료 후 결과 메시지 확인

## 입력 예시

### 텍스트 입력 예시:
```
323
2134
3212
4567
```

### 파일명 매칭 예시:
- 입력: `323`
- 매칭 파일: `abc323.jpg`, `photo323.png`, `file323.txt` 등

## 주의사항

- 파일명에 입력한 숫자가 포함되어 있어야 매칭됩니다
- 여러 파일이 같은 숫자를 포함하고 있으면 첫 번째로 찾은 파일만 복사됩니다
- 저장 폴더에 같은 이름의 파일이 있으면 자동으로 이름이 변경됩니다 (예: `file.txt` → `file_1.txt`)

## 시스템 요구사항

- Python 3.6 이상
- tkinter (Python 기본 포함)
- Windows, macOS, Linux 지원

## 문제 해결

### 파일을 찾을 수 없다는 메시지가 나올 때:
1. 원본 폴더 경로가 올바른지 확인
2. 파일명에 입력한 숫자가 정확히 포함되어 있는지 확인
3. 대소문자 구분 확인

### 복사가 실패할 때:
1. 저장 폴더에 쓰기 권한이 있는지 확인
2. 디스크 용량이 충분한지 확인
3. 파일이 다른 프로그램에서 사용 중이 아닌지 확인

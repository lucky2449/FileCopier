# 파일 복사 프로그램

숫자, 파일명, 경로 목록을 입력하면 원본 폴더에서 대상 폴더로 파일을 자동 복사해주는 Windows 프로그램입니다.

## 다운로드

**[⬇ 지금 설치하기](https://lucky2449.github.io/FileCopier/)**

또는 [직접 다운로드 (FileCopier_download.exe)](https://github.com/lucky2449/FileCopier/raw/main/installer_output/FileCopier_download.exe)

> 설치 후 Windows 보안 경고가 뜨면 **"추가 정보" → "실행"** 을 클릭하세요.

---

## 사용 방법

1. **원본 폴더** 선택 — 복사할 파일이 있는 폴더
2. **저장 폴더** 선택 — 파일을 복사할 대상 폴더
3. **확장자 선택** — 원본 폴더 선택 시 자동 스캔됨. 원하는 확장자 또는 "모든 파일" 선택
4. **목록 입력** — 숫자, 파일명, 전체 경로를 입력 (공백, 줄바꿈, 쉼표, 슬래시 모두 가능)
5. **파일 복사 시작** 클릭

### 입력 예시

```
323
2134
abc1234.jpg
C:\photos\DSC00042.ARW
```

### 목록 불러오기

| 방법 | 설명 |
|---|---|
| 직접 입력 | 텍스트 영역에 항목을 직접 입력 |
| TXT 파일 불러오기 | `.txt` 파일을 불러와 목록 자동 채우기 |
| 파일 다이얼로그 | 파일 선택 창으로 직접 파일 추가 |

---

## 파일 매칭 방식

입력값 하나당 아래 순서로 파일을 탐색합니다:

1. **절대 경로** — 경로가 그대로 존재하면 해당 파일 사용
2. **파일명 일치** — 확장자 포함 파일명이 정확히 일치
3. **스템 일치** — 확장자를 제외한 파일명이 일치
4. **숫자 접미사 일치** — 파일명 끝 숫자와 입력 숫자 일치 (예: `323` → `DSC00323.jpg`)

중복 파일은 자동으로 이름을 변경합니다 (예: `file.jpg` → `file_1.jpg`)

---

## 문제 해결

**파일을 찾지 못하는 경우**
- 원본 폴더 경로가 올바른지 확인
- 확장자 필터 설정 확인
- 입력한 숫자가 파일명에 실제로 포함되어 있는지 확인

**복사가 실패하는 경우**
- 저장 폴더에 쓰기 권한이 있는지 확인
- 디스크 여유 공간 확인

## 개발자용

### Python으로 직접 실행

```bash
pip install -r requirements.txt
python modern_file_copier.py
```

### EXE 빌드

```bash
python utils/build_exe.py
```

`dist/FileCopier.exe` 단일 파일이 생성됩니다.

### 시스템 요구사항

- Windows 7 이상 (EXE 실행 시 Python 불필요)
- Python 3.8 이상, PyQt5 >= 5.15.0 (직접 실행 시)

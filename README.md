# 파일 복사 프로그램

숫자, 파일명, 파일 경로 목록을 입력받아 소스 폴더에서 대상 폴더로 파일을 일괄 복사하는 PyQt5 GUI 애플리케이션입니다. 주로 숫자 ID로 이미지/미디어 파일을 복사하는 용도로 사용됩니다.

## 실행 방법

### EXE 실행 (권장)

`dist/FileCopier.exe` 를 더블클릭하여 실행합니다.

> Windows SmartScreen 경고가 나타나면 "추가 정보" → "실행" 을 클릭하세요.

### Python으로 직접 실행

```bash
pip install -r requirements.txt
python modern_file_copier.py
```

## 사용 방법

1. **원본 폴더** — 복사할 파일이 있는 폴더를 선택합니다.
2. **저장 폴더** — 파일을 복사할 대상 폴더를 선택합니다.
3. **확장자 선택** — 원본 폴더를 선택하면 자동으로 확장자를 스캔합니다. 원하는 확장자를 선택하거나 "모든 파일"을 선택합니다.
4. **목록 입력** — 텍스트 영역에 숫자, 파일명, 또는 전체 경로를 입력합니다. 구분자는 공백, 줄바꿈, 쉼표, 슬래시 모두 허용됩니다.
5. **파일 복사 시작** 버튼을 클릭합니다.

### 입력 방법

| 방법 | 설명 |
|---|---|
| 직접 입력 | 텍스트 영역에 항목을 직접 입력 |
| TXT 파일 불러오기 | `.txt` 파일을 불러와 목록 자동 채우기 |
| 파일 다이얼로그 | 파일 선택 다이얼로그로 직접 파일 추가 |

### 입력 예시

```
323
2134
abc1234.jpg
C:\photos\DSC00042.ARW
```

## 파일 매칭 로직

각 입력 항목은 다음 순서로 파일을 탐색합니다:

1. **절대 경로** — 경로가 그대로 존재하면 해당 파일 사용
2. **파일명 일치** — 확장자 포함 파일명이 정확히 일치
3. **스템 일치** — 확장자를 제외한 파일명이 일치
4. **숫자 접미사 일치** — 스템의 끝 숫자(앞의 0 제거)와 입력 숫자가 일치 (예: `323` → `DSC00323.jpg`)

## 빌드

```bash
python utils/build_exe.py
```

`dist/FileCopier.exe` 단일 파일이 생성됩니다.

## 자동 업데이트

앱 시작 시 GitHub Releases API를 백그라운드에서 확인하여 새 버전이 있으면 상단에 업데이트 배너를 표시합니다. `utils/updater.py` 상단의 `GITHUB_REPO` 값을 실제 저장소 주소로 설정해야 동작합니다.

```python
GITHUB_REPO = "your-username/your-repo"
```

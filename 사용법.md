# 파일 복사 프로그램 사용법

## 📁 프로그램 소개
숫자 목록을 입력하여 해당하는 파일들을 자동으로 복사하는 프로그램입니다.

## 🚀 실행 방법

### 1. 실행 파일 사용 (권장)
- `dist` 폴더에 있는 `파일복사프로그램.exe` 파일을 더블클릭하여 실행
- **주의**: Windows Defender가 처음 실행 시 경고를 표시할 수 있습니다.
  - 이는 정상적인 현상입니다 (새로 생성된 실행 파일에 대한 보안 검사)
  - "추가 정보" → "실행" 클릭하여 실행하세요

### 2. Python으로 직접 실행
```bash
python modern_file_copier.py
```

## 📋 사용 방법

### 1단계: 경로 설정
- **원본 폴더**: 복사할 파일들이 있는 폴더 선택
- **저장 폴더**: 파일을 복사할 대상 폴더 선택

### 2단계: 파일 확장자 선택
- 원본 폴더 선택 시 자동으로 파일 확장자를 스캔합니다
- 원하는 확장자를 선택하거나 "모든 파일" 선택

### 3단계: 숫자 목록 입력
**직접 입력 방법:**
- 텍스트 영역에 숫자를 입력
- 예시: `1, 10, 13, 16` 또는 `1 10 13 16`

**텍스트 파일 방법:**
- "텍스트 파일 업로드" 선택
- "텍스트 파일 로드" 버튼 클릭하여 .txt 파일 업로드

### 4단계: 파일 복사 실행
- "🚀 파일 복사 시작" 버튼 클릭
- 진행 상황을 실시간으로 확인

## 🎯 주요 기능

### 스마트 파일 검색
- 파일명에 숫자가 포함된 파일을 자동으로 찾습니다
- 예: `DSC001.RWA`, `photo123.jpg` 등

### 다양한 구분자 지원
- 쉼표: `1, 10, 13`
- 슬래시: `1 / 10 / 13`
- 공백: `1 10 13`
- 줄바꿈으로 구분된 숫자들

### 자동 확장자 스캔
- 원본 폴더 선택 시 자동으로 파일 확장자를 스캔
- 하나만 있으면 자동 선택, 여러 개면 사용자가 선택

### 중복 파일 처리
- 같은 이름의 파일이 있으면 자동으로 이름 변경
- 예: `file.txt` → `file_1.txt`

## ⚠️ 주의사항

### Windows Defender 경고
실행 파일을 처음 실행할 때 Windows Defender가 경고를 표시할 수 있습니다:

1. **SmartScreen 필터 경고**
   - "Windows에서 PC를 보호했습니다" 메시지가 나타날 수 있습니다
   - "추가 정보" 클릭 → "실행" 클릭

2. **바이러스 검사 경고**
   - 일부 백신 프로그램에서 경고를 표시할 수 있습니다
   - 이는 PyInstaller로 생성된 실행 파일의 일반적인 현상입니다
   - 프로그램을 신뢰할 수 있는 경우 예외 처리하세요

### 해결 방법
1. **실행 파일을 신뢰할 수 있는 파일로 추가**
   - Windows Defender → 바이러스 및 위협 방지
   - 제외 항목에 `파일복사프로그램.exe` 추가

2. **관리자 권한으로 실행**
   - 실행 파일을 우클릭 → "관리자 권한으로 실행"

## 📊 시스템 요구사항
- **운영체제**: Windows 7 이상
- **메모리**: 최소 100MB 여유 공간
- **Python**: 실행 파일 사용 시 불필요 (포함됨)

## 🔧 문제 해결

### 프로그램이 실행되지 않는 경우
1. Windows Defender 예외 처리
2. 관리자 권한으로 실행
3. Python 버전 확인 (직접 실행 시)

### 파일을 찾을 수 없다는 메시지
1. 원본 폴더 경로 확인
2. 파일명에 입력한 숫자가 정확히 포함되어 있는지 확인
3. 확장자 필터 설정 확인

### 복사가 실패하는 경우
1. 저장 폴더에 쓰기 권한 확인
2. 디스크 용량 확인
3. 파일이 다른 프로그램에서 사용 중인지 확인

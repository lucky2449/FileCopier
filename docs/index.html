파일복사 프로그램

숫자 목록만 입력하면
해당 파일을 자동으로 찾아 복사해 드립니다.

⬇ 지금 설치하기 (Windows)

Windows 7 이상 · 관리자 권한 불필요 · 최신 버전 자동 업데이트

주요 기능

  • 숫자, 파일명, 경로를 자유롭게 입력
  • 원본 폴더에서 대상 폴더로 일괄 복사
  • 확장자 필터로 원하는 파일만 선택
  • 중복 파일 자동 이름 변경
  • 새 버전 출시 시 앱에서 자동 알림
⚠ 설치 시 Windows 보안 경고가 뜰 수 있어요 "Windows에서 PC를 보호했습니다" 메시지가 나타나면
"추가 정보"를 클릭한 후 "실행"을 클릭하시면 됩니다.

설치 방법

  1. 위 버튼을 눌러 설치 파일을 다운로드합니다.
  2. 다운로드된 FileCopier_설치_vX.X.X.exe를 더블클릭합니다.
  3. 보안 경고가 뜨면 "추가 정보 → 실행"을 클릭합니다.
  4. 설치 버튼을 클릭하면 완료됩니다.
